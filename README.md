# Software de Vendas de Bicicletas Elétricas - README

## Introdução

Este documento define os requisitos e funcionalidades do software de vendas de bicicletas elétricas, denominado BIKE.

## Objetivo

Definir os requisitos e funcionalidades do software de venda de bicicletas elétricas.

## Escopo

O software abrangerá desde o cadastro de produtos até o fechamento de vendas, incluindo gestão de estoque e informações de clientes.

## Definições

- **BIKE**: Software de Venda de Bicicletas Elétricas
- **CRUD**: Create, Read, Update, Delete (Criar, Ler, Atualizar, Excluir)

## Funcionalidades

- **Gerenciamento de produtos**: Cadastro, edição, visualização e exclusão de bicicletas elétricas.
- **Realização de vendas**: Cálculo de preços, atualização de estoque e complementação da transação de compra.
- **Cadastro de clientes**: Coleta de informações como nome, telefone, etc.

## Integrações

- **Sistemas de pagamento online**
- **Sistemas de gestão de estoque**

## Prioridades

- **Alta prioridade**: CRUD de produtos.
- **Média prioridade**: Processo de venda e transação de compra.

## Requisitos Não Funcionais

- **Desempenho**: Suporte a um grande número de transações simultâneas durante picos de vendas.
- **Usabilidade**: Interface intuitiva e fácil de usar.
